SQA-Internship-Project-Task
Software Quality Assurance (SQA) internship project focusing on manual testing, bug tracking, and test automation for web applications.
Project Overview
This repository contains the deliverables for Week 1 of my Software Quality Assurance (SQA) internship at DevelopersHub Corporation. The focus for this week was on Manual Testing fundamentals, Test Case design, and Bug Reporting.

Tasks Completed 
SDLC & STLC Analysis: Studied the integration of QA processes within the Software Development Life Cycle.
Test Case Design: Authored 5 comprehensive test cases for a Login/Signup system using The Internet (Herokuapp).
Manual Testing: Conducted functional and UI testing to identify system defects.
Bug Tracking: Documented and reported identified bugs using Jira, ensuring all reports include Severity, Priority, and Steps to Reproduce.

Files in this Folder
SQA_Week1_Task_TestCases: A detailed document including Test IDs, Descriptions, Preconditions, and Execution Status.
SQA_Week1_Task_BugReport: A professional bug report sheet exported from Jira covering functional and UI failures.

Tools Used
Jira: For defect logging and management.
Google Sheets/Word: For test case documentation.
Chrome DevTools: For cross-browser and mobile UI testing.
